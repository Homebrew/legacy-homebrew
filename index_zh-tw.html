Homebrew

OS X 缺少的套件管理工具

English Version | 简体中文版 | 繁體中文版

  • 使用 Homebrew 安裝 Apple 沒有預裝但是你需要的東西。

    $ brew install wget

  • Homebrew 會將 packages 安裝在他們自己的目錄，然後把檔案 symlink 到 /usr/local 下。

    $ cd /usr/local
$ find Cellar
Cellar/wget/1.12
Cellar/wget/1.12/bin/wget
Cellar/wget/1.12/share/man/man1/wget.1

$ ls -l bin
bin/wget -> ../Cellar/wget/1.12/bin/wget

  • Homebrew 不會把檔案放在預設路徑之外的地方，因此可以在任何位置使用 Homebrew 安裝程式。

  • 輕鬆建立你自己的 Homebrew packages。

    $ brew create http://foo.com/bar-1.0.tgz
Created /usr/local/Library/Formula/bar.rb

  • 完全以 Git 和 Ruby 為基底，所以你可以盡情地運用這些知識，輕鬆地復原你的修改以及合併上游的更新。

    $ brew edit wget # opens in $EDITOR!

  • Homebrew 的 formula 都是簡單的 Ruby 腳本：

    require 'formula'

class Wget < Formula
  homepage 'http://www.gnu.org/wget/'
  url 'http://ftp.gnu.org/wget-1.12.tar.gz'
  md5 '308a5476fc096a8a525d07279a6f6aa3'

  def install
    system "./configure --prefix=#{prefix}"
    system 'make install'
  end
end

  • Homebrew 互補了 OS X，你可以使用 gem 來安裝 Ruby 套件， 而它的依存軟體可以用 brew 安裝。

  • 安裝 Homebrew

    ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"

    在終端機命令列提示貼上這個。

    腳本執行時會解釋它正在做什麼，並在你確認之前暫停下來。你可以在這裡找到更多安裝選擇（需要 10.5 以上版本）。
