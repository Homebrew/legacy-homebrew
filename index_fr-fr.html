Homebrew

Le gestionnaire package pour OS X

English Version | Version Francaise | 简体中文版 | 繁體中文版

  • Homebrew installe ce dont vous avez besoin et qu'Apple n'a pas installé.

    $ brew install wget

  • Homebrew installe les packages dans leurs propres répertoires et symlinks leurs fichiers vers /usr/local.

    $ cd /usr/local
$ find Cellar
Cellar/wget/1.12
Cellar/wget/1.12/bin/wget
Cellar/wget/1.12/share/man/man1/wget.1

$ ls -l bin
bin/wget -> ../Cellar/wget/1.12/bin/wget

  • Homebrew n'installera pas de fichiers en dehors de son préfixe, et vous pouvez placer Homebrew où vous le désirez.

  • Créez facilement vos propre packages Homebrew.

    $ brew create http://foo.com/bar-1.0.tgz
Created /usr/local/Library/Formula/bar.rb

  • Homebrew utilise git et ruby, vous pouvez donc faire des modifications sans crainte, sachant que vous pourrez facilement les annuler et les intégrer avec les mises à jour future.

    $ brew edit wget # opens in $EDITOR!

  • Les formules Homebrew sont de simples scripts Ruby:

    require 'formula'

class Wget < Formula
  homepage 'http://www.gnu.org/wget/'
  url 'http://ftp.gnu.org/wget-1.12.tar.gz'
  md5 '308a5476fc096a8a525d07279a6f6aa3'

  def install
    system "./configure --prefix=#{prefix}"
    system 'make install'
  end
end

  • Homebrew est un complément pour OS X. Installez vos gems avec gem, et leurs dépendances avec brew.

  • Installer Homebrew

    ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"

    Copiez et collez dans une fenêtre du Terminal.

    Le script explique ce qu'il va faire, puis fait une pause avant de l'exécuter. Plus d'options d'installation sont disponibles ici (Requis sur 10.5).
